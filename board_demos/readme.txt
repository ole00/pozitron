This directory contains basic demos for STM32F103VET6 CPU.
The purpose of the demos is to test basic functionality
of the development board and your build environment.

These demos don't rely on uzebox kernel, but need its
build scripts to compile the demo binaries.

How to build and run:
- enter the testXX/default directory
- type 'make clean all' to build binaries
- type 'sudo make burn' to flash the dev board
- press Reset buton on the dev board to run the flashed binary
